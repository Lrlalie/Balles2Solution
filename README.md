# Balles2Solution

## DESCRIPTION

* Application qui génère un mouvement perpétuel que l'on peut appliquer à un objet ou plusieurs.
* Dans notre exemple : 2 balles.

## INSTALLATION

* Clone this repository
* git clone  https://github.com/Lrlalie/Balles2Solution
